# Justice for Julian

A national advocacy campaign website seeking accountability for preventable hospital failures.

## Setup

### 1. Add Photos
Place these two images in the `images/` folder:
- `julian-life.jpg` — Happy photo of Julian (used on left side of hero)
- `julian-hospital.jpg` — Hospital photo (used on right side of hero)

### 2. Deploy on GitHub Pages
1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to **Deploy from a branch** → `main` → `/ (root)`
4. The site will be live at `https://yourusername.github.io/justice-for-julian/`

### 3. Custom Domain (optional)
1. In **Settings → Pages → Custom domain**, enter your domain
2. Add a CNAME record pointing to `yourusername.github.io`

## Files
- `index.html` — Version A (formal, Playfair Display + Inter)
- `version-b.html` — Version B (warmer, Cormorant Garamond + DM Sans)
- `images/` — Hero photos
